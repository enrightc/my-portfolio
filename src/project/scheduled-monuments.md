## overview

A small data visualisation project built to explore how open heritage datasets
can be cleaned, processed, and brought to life on an interactive map. Using a
CSV of Scheduled Monuments in Wales sourced from Cadw, the data was processed
in Pandas and visualised as a time-based, interactive map in Plotly.

A first step into data processing and geospatial visualisation, and a reminder
that even simple datasets can tell interesting stories when mapped well.

---

## what I learned

- How to clean and process real-world heritage data using **Pandas**, handling
  inconsistencies and preparing it for visualisation
- How to create an interactive, time-based map using **Plotly**, exploring how
  data can be explored visually rather than just read from a table
- How open government datasets — such as Cadw's Designated Historic Asset GIS
  data — can be repurposed to tell compelling geographic stories
