## overview

A Welsh archaeology-themed memory card game built as a side project to
practise JavaScript and CSS Flexbox skills. Players flip cards to find
matching pairs, with each card featuring an iconic Welsh heritage site.

A fun, lightweight project that combines a love of Welsh archaeology with
front-end development.

---

## key features

- **Memory card gameplay**  
  Classic flip-and-match mechanics built entirely in vanilla JavaScript.

- **Welsh heritage theme**  
  Cards feature iconic archaeological sites from across Wales, giving the
  game a distinctive and personal feel.

- **Flexbox layout**  
  The card grid is built using CSS Flexbox, keeping the layout clean and
  structured.

---

## what I learned

- How to manage game state in vanilla JavaScript — tracking flipped cards,
  matched pairs, and preventing clicks during card animations
- How to use **CSS Flexbox** to build a responsive, evenly spaced card grid
- That small side projects are a great way to consolidate skills and build
  something genuinely enjoyable in the process
